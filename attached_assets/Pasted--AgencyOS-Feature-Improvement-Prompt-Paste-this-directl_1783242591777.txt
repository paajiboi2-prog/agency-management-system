## 3. EXCEL EXPORT + SPREADSHEET VIEW (Finance Section)

### A. Excel Export Button

Add an **"Export to Excel"** button in the Finance page header (visible in all tabs).

```jsx
// Use the 'xlsx' (SheetJS) library
// npm install xlsx

import * as XLSX from 'xlsx';

function exportToExcel(data, sheetName, filename) {
  const ws = XLSX.utils.json_to_sheet(data);
  const wb = XLSX.utils.book_new();
  XLSX.utils.book_append_sheet(wb, ws, sheetName);

  // Auto-width columns
  const colWidths = Object.keys(data[0] || {}).map(key => ({
    wch: Math.max(key.length, ...data.map(row => String(row[key] ?? '').length))
  }));
  ws['!cols'] = colWidths;

  XLSX.writeFile(wb, filename);
}

// Usage per tab:
// Invoices:  exportToExcel(invoices, 'Invoices', 'AgencyOS_Invoices.xlsx')
// Quotations: exportToExcel(quotations, 'Quotations', 'AgencyOS_Quotations.xlsx')
// etc.

// Button placement: top-right of Finance page header
// Icon: FileSpreadsheet from lucide-react
// Label: "Export Excel"
// Variant: outline button
```
